# Práctica de Java Swing C02 - Gestión de apartamentos Turísticos.

Proyecto Comprendio de desarollo con Java Swing para 2ºDAM.
Formularios, validaciones, resumen y cálculo de precio automático.

## ejecución
Windows:
- Doble click sobre el archivo .jar

Línea de comandos (Windows / Linux):
- situarse en el directorio del archivo
- ejecutar: java -jar nombre.jar

En Linux, si no tiene permisos:
- chmod +x nombre.jar
- java -jar nombre.jar

## Tecnologías
Java Swing, IDE: Eclipse
